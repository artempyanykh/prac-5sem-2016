Задание № 1

Задание: 

Решение матричной игры путем сведении ее к паре двойственных задач линейного программирования и визуализации спектров оптимальных 
         стратегий. 

Решение:  

Использовались функция Linprog из библиотеки Scipy и алгоритм  из книги "Исследование операций" (описанной в разделе 6.4.3.), 
          а для визуализации - библиотека Matplotlib.

Операционая система: Linux (ubuntu).

Запуск: 

        1. Необходимо с консоли запустить jupyter notebook.

        2. Выбирать директорию, содержащую task1.ipynb. 
        
        3. Запускаем ее.
