1. В задании требовалось написать программу, которая бы решала матричную антогонистическую игру и 
визуализировать ее решение. Матричная игра сводилась к двум задачам линейного программирования,
которые решалиь через специальную функцию модуля Scipy: linprog. Далее, при помощи модуля
matplotlib строились гистограммы решений.

2. ОС Ubuntu. ctrl + alt + T -> запускается терминал. В нем запускается jupyther notebook путем 
вызова в терминале команды ipython notebook. В открывшемся браузере можно открыть проект под назавнием 
matrix_game.ipynb, который можно будет запустить.
